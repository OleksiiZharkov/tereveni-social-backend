## 📖 Overview
**Tereveni** is a robust backend infrastructure for a social media platform, developed as a comprehensive team project. It provides a highly scalable RESTful API for user management, post creation, and secure interactions, following modern backend engineering principles.

## 👨‍💻 My Role & Contributions
This API was built collaboratively in a team environment. As a Core Backend Developer, my specific responsibilities included:
* **Database Architecture:** Designed, normalized, and configured the **PostgreSQL** schema for optimal data storage and retrieval.
* **Security Implementation:** Engineered a stateless authentication and authorization system utilizing **JSON Web Tokens (JWT)** and Spring Security.
* **Core REST APIs:** Developed and optimized secure endpoints for user registration, authentication (login), and profile management.

## 🛠 Tech Stack
* **Language:** Java 11/17
* **Framework:** Spring Boot, Spring MVC, Spring Data JPA
* **Database:** PostgreSQL
* **Security:** Spring Security, JWT
* **Build Tool:** Maven
* **Version Control:** Git & GitHub

## 🚀 Key Features
* **Stateless Authentication:** Secure login and registration using JWT to ensure API scalability.
* **User Management:** Full CRUD operations for user profiles with role-based access control.
* **Data Persistence:** Optimized relational data mapping using Hibernate/JPA.
* **Email Integration:** Automated email notification service for user onboarding.
